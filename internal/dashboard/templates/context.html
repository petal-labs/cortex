Context Store

The context store holds key-value pairs for maintaining state across conversations. Context values are accessed by specific keys through the MCP tools.

Context listing requires specific key lookup

Use the MCP tools to get/set context values by key

Usage

Available context operations:

  • context_set - Store a value with a key
  • context_get - Retrieve a value by key
  • context_delete - Remove a key-value pair
  • context_search - Search context by semantic similarity
